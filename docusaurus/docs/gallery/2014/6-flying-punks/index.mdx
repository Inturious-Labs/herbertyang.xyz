# 飞扬的骚年

2009年8月，首届张北草原音乐节，向四十年前的Woodstock致敬，创造了当时中国室外音乐节参加人数的记录。在北京河北交界之处的张北，有一望无际的草原，有三天无休的摇滚，还有几个飞扬的骚年。 欧耶！

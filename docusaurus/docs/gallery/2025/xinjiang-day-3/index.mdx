- 前一天：[金秋喀什 - 阿克苏](https://herbertyang.xyz/docs/gallery/2025/xinjiang-day-2/)

南疆地域极其宽广，每个主要景点之间的开车时间都是六个小时起步。经过沙漠里长途跋涉和小心翼翼的让车，我们的大巴终于来到了位于塔克拉玛干沙漠边缘，北纬39度的`盖买提N39沙漠营地`。这里是国内顶级沙漠汽车拉力赛的主战场，但今天人不多，只有我们还有其他几个团的游客。

大巴开进停车场的时候，离落日时间 20:58 还有两分钟，夕阳依旧怒射，已经有一半跌落在云里。等到我以最快速度下车爬上最近的沙坡时，太阳竟然就在这一转眼之间完全没入云海，踪影全无，连一丝余晖都没有留下来，这也太急匆匆了吧。

晚饭后，营地的领队带着大家走入沙漠深处。不打灯光，不用手机，纯粹在月光指引下，锦衣夜行地来到一座剧院般的巨大沙丘。我们躺下来，倾听流沙的声音。一丝风也没有，只有半轮雪月高悬黑沉沉的夜空。远处渐渐飘来悠扬的排箫声，西域传奇，`十二木卡姆`，登场了。

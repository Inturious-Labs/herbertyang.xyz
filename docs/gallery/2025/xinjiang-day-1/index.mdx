- 后一天：[金秋喀什 - 阿克苏](https://herbertyang.xyz/docs/gallery/2025/xinjiang-day-2/)

十一长假期间参加般马旅行团组织的`金秋喀什`之行。第一天在库车集合。库车古称龟兹（终于知道这个词怎么念了），以前西域民族大融合的重镇，类似于现在的小商品贸易宇宙中心乌镇。

出租车司机抱怨现在的天气越来越热，冬天已经不下雪了。路过城区的几座桥，河道都干涸了。看来不光上海越来越热，欧亚大陆的连接纽带喀什地区也是如此。

库车的游人并不多，可能晚上才会更加热闹一些吧。四百多年历史的库车大寺，颇有点西班牙阿尔罕伯拉宫的感觉。

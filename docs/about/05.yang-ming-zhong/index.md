# 杨明忠

## 简介

1943年8月9日出生。

1961年9月至1966年7月在华中工学院(现华中科技大学)动力机械专业本科毕业。

1966年9月至1973年9月在电力工业部邯峰安电业局从事机械设计和制造工作。

1973年10月起在(武汉汽车工业大学)武汉理工大学机电工程学院任教。1986年起任教授。

1991年起享受国务院政府特殊津贴。

1990年被国务院评为机械设计及理论专业博士导师(武汉汽车工业大学第一位博士导师)。

1989—2002年任武汉汽车工业大学、武汉理工大学机电工程学院院长。

1979年11月至1982年12月在意大利的ISPRA研究中心和ASSORENI研究所做访问学者。

1990年8月至1991年6月在美国的密歇根理工大学做高级访问学者。

主要研究方向：
 - 现代设计理论与CAD/CAM集成技术
 - 机械系统检测与控
 - 先进制造技术与企业信息化技术

## 主要科研成果

负责并完成：
- 两项国家经委重点科技攻关项目
- 一项国家科学技术部中法先进研究计划项目
- 一项国家自然科学基金项目
- 四项机械部科技攻关项目
- 七项湖北省科委科技项目
- 四项武汉市科委科技项目和30多项企业科技项目
- 在《Wear》等学术刊物上发表学术论文186篇，其中SCI收录10篇，EI收录19篇，ISTP收录15篇。
- 主编《摩擦学设计基础》（普通高等教育机电类规划教材），机械工业出版社出版，1996年6月再版
- 主编《机械设计》(教育部高等教育面向21世纪课程教材，普通高等教育机电类规划教材)，2001年机械工业出版社出版
- 主编《MACHINERY DESIGN》(英语版)(教育部高等教育面向21世纪课程教材，普通高等学校机械工程类专业双语系列英语教材)，2004年武汉理工大学出版社出版，2006年再版
- 2016年参入负责完成国家工程院重大项目：2025中国创新设计制造路线图

## 荣誉

- 1995年获全国机械工业先进工作者称号
- 1991年被国家教委和国家人事部授予“在祖国社会主义现代化建设中做出突出贡献的回国留学人员”称号
- 1988年获全国汽车工业劳动模范称号
- 2004年获湖北省优秀研究生导师称号
- 已指导培养并授于学位的博士17名和硕士38名。
- 作为第一负责人获湖北省科技进步二等奖和三等奖各一项
- 机械工业部科技进步三等奖一项
- 武汉市科技进步二等奖一项

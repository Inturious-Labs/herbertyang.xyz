# 鱼香肉丝

本季最为惊艳的作品，做了两次，都很成功。也是需要切无数的菜，但很值得，看来会成为杨记的保留主菜之一了。

![鱼香肉丝](./img/cookbook-yu-xiang-rou-si.jpg)

1. 黑木耳用温水泡发开洗净
2. 猪里脊肉解冻后顺势切成丝，用一小撮盐抓捏到发粘。加入水淀粉（淀粉：水=1：1），加入肉丝中用手抓捏到全部吸收。最后加一汤匙色拉油和料酒，拌匀后腌制待用。
3. 冬笋切片后入开水锅中焯水，捞出来后切成丝
4. 胡萝卜去皮切丝
5. 泡好的黑木耳切丝
6. 准备好姜末，葱末，蒜末
7. 一汤匙干淀粉加三汤匙清水调成水淀粉；把料酒，醋，生抽，白糖，香麻油和水调成调味汁
8. 热锅放油，在低油温时倒入肉丝大火煸炒到肉丝表面变色，盛出备用
9. 用锅中余油煸炒姜蒜末小火炒出香味，再倒入剁椒一起炒出香味
10. 倒入冬笋丝，胡萝卜丝，和黑木耳丝转大火翻炒一会儿，再倒入肉丝
11. 倒入准备好的调味汁，翻炒一分钟
12. 慢慢倒入水淀粉，边倒边炒，直到汤汁到达合适的浓稠度，关火。撒上葱花，出锅。
